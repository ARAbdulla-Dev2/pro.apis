API Endpoints

YouTube Audio

Extract audio streams from YouTube videos

GET /api/ytmp3?url={youtube_url}&apiKey=YOUR_KEY

YouTube Video

Get video+audio streams from YouTube

GET /api/ytmp4?url={youtube_url}&apiKey=YOUR_KEY

Facebook Video

Extract videos from Facebook posts

GET /api/fb?url={facebook_url}&apiKey=YOUR_KEY

Instagram Media

Download videos and reels from Instagram

GET /api/ig?url={instagram_url}&apiKey=YOUR_KEY

TikTok Video

Extract videos from TikTok with metadata

GET /api/tk?url={tiktok_url}&apiKey=YOUR_KEY

API Documentation

Authentication

All endpoints require an API key passed as a query parameter:

?apiKey=YOUR_API_KEY

Response Format

Successful responses return JSON data:

{
    "success": true,
    "data": {
        "title": "Video Title",
        "thumbnail": "thumbnail_url",
        "formats": [
            {
                "url": "download_url",
                "quality": "hd",
                "type": "video/mp4"
            }
        ]
    }
}

Error Handling

Errors return appropriate HTTP status codes:

{
    "error": "Error message",
    "details": "Additional error details"
}
