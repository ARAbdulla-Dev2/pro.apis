ARAbdulla-Dev API

Powerful media extraction API services for YouTube, Facebook, Instagram, and TikTok

Available Endpoints

YouTube Audio

Extract audio streams from YouTube videos in various formats.

GET /api/ytmp3?url={youtube_url}&apiKey=YOUR_KEY

YouTube Video

Get video+audio streams from YouTube with multiple quality options.

GET /api/ytmp4?url={youtube_url}&apiKey=YOUR_KEY

Facebook Video

Extract video content from Facebook posts and reels.

GET /api/fb?url={facebook_url}&apiKey=YOUR_KEY

Instagram Media

Download videos and reels from Instagram posts.

GET /api/ig?url={instagram_url}&apiKey=YOUR_KEY

TikTok Video

Extract videos from TikTok with metadata and direct download links.

GET /api/tk?url={tiktok_url}&apiKey=YOUR_KEY

API Documentation

Authentication

All endpoints require an API key passed as a query parameter:

?apiKey=YOUR_API_KEY

Response Format

Successful responses return JSON data with the following structure:

{
    "success": true,
    "data": {
        "title": "Video Title",
        "thumbnail": "thumbnail_url",
        "formats": [
            {
                "url": "download_url",
                "quality": "hd",
                "type": "video/mp4"
            }
        ]
    }
}

Error Handling

Errors return appropriate HTTP status codes with JSON responses:

{
    "error": "Error message",
    "details": "Additional error details if available"
}
